Example code

This code defines a function called greet. When the function is called, it prompts the user to enter their name using a pop-up window and stores the input in the name variable. Then, it creates a new Date object and assigns it to the date variable to represent the current date and time. The toLocaleDateString() method is called on the date object to convert the date to a string representation based on the user's locale. The resulting string is stored in the today variable. Finally, an alert message is displayed using template literals, which includes the user's name and the formatted current date. 

            
              function greet() {
                const name = prompt('Enter your name:');
                const date = new Date();
                const today = date.toLocaleDateString();
                alert(`Hello, ${name}! Today is ${today}.`);
              }
              
              greet();
